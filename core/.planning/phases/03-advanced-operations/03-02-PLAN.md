       - On success -> return PresignedUploadResponse(storageKey=key, uploadUrl=url, method="PUT", supported=true)

       `confirmPresignedUpload(workspaceId: UUID, request: ConfirmUploadRequest): UploadFileResponse`
       - @PreAuthorize for workspace access
       - val userId = authTokenService.getUserId()
       - Verify file exists: `require(storageProvider.exists(request.storageKey)) { "File not found at storage key" }` -- or throw StorageNotFoundException if exists returns false
       - Download file for validation: `val downloadResult = storageProvider.download(request.storageKey)`
       - Read bytes, detect content type via Tika: `contentValidationService.detectContentType(stream, request.originalFilename)`
       - Extract domain from storage key (the key format is `{workspaceId}/{domain}/{uuid}.{ext}`) OR require domain in the confirm request. Decision: parse domain from key prefix to keep API simple.
       - Validate content type against domain allowlist. If invalid: delete file from provider via `storageProvider.delete(request.storageKey)`, then throw ContentTypeNotAllowedException
       - Validate metadata if provided: `request.metadata?.let { validateMetadata(it) }`  (validate as non-null map of String to String)

       - Log upload activity
       - Generate signed download URL
       - Return UploadFileResponse

       - Validate the incoming metadata: call validateMetadata but allow null values (the UpdateMetadataRequest has Map<String, String?> -- null values mean remove)
       - Merge: `val merged = mergeMetadata(entity.metadata, request.metadata)`
       - entity.metadata = if (merged.isEmpty()) null else merged
       - Save entity
       - Log FILE_UPDATE activity with details: "previousMetadata" to (old metadata), "updatedMetadata" to (new metadata)
       - Return entity.toModel()

       Private helper `mergeMetadata(existing: Map<String, String>?, patch: Map<String, String?>): Map<String, String>`:

       - patch.forEach { (key, value) -> if (value == null) merged.remove(key) else merged[key] = value }

       - Validate each non-null value length <= 1024
