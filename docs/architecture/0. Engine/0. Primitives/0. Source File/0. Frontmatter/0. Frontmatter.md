## Example: Full Source Shape + Frontmatter 

The on-disk representation a source file serializes to (illustrative, not normative):

```yaml
---
source_id: <uuid-v5>
canonical_uri: <permalink-or-cranium-urn>
display_name: <string>
source_class: live_pointer | native | hybrid | digest | reflection
sor_kind: external | cranium | external+cranium | none
sor_pointer: <opaque per-lane reference | null>
write_authority: <enum + actor list>
lane: <lane slug>
lane_metadata: { ... }
ingest_actor: <connector or user id>
confidence: <float | "personal">
hash_skip_eligible: <bool>
drift_signal_kind: rehash_diff | edit_event | dual | none
drift_subscriber: <topic | null>
first_ingested_at: <iso8601>
last_synced_at: <iso8601>
last_drift_event_at: <iso8601 | null>
sync_status: ok | stale | broken_citation | class_mismatch
pages_derived: [<page_id>, ...]
active: <bool>
frozen_reason: <enum | null>

# class-specific block (one of):
upstream_ref: { ... }           # live_pointer, hybrid
authored_by: <user_id>          # native, digest, reflection
parents: [ ... ]                # digest only
owner: <user_id>                # reflection only
visibility: <enum>              # reflection only

regions:
  - region_id: <slug>
    locator: { kind, value }
    region_class: upstream | native
    content_hash: <sha256>
    last_hashed_at: <iso8601>
    cited_by: [<page_id>, ...]
---

<optional highlights / SME notes body — for native/digest/reflection only>
```
