### `Overview`
Digests are essentially summaries and finings that have been produced as a result of agentic web research about a particular topic or domain to better improve context and internal wiki knowledge with external sources of information

### `Confidence`
### `System of Record`
Given that this is derived from web research. Findings will be extracted from numerous related sources of information
	- websites
	- journals/blogs
	- research papers
	- videos
Given that this information will then be summarised, and findings will be aggregated to represent patterns and trends based on the topic that was researched, there is no pure system of record for the digest. Instead is made up of a culmination of all the different sources.
