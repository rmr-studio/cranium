  <action>
    1. Create `StorageController` in `controller/storage/`:
       - `@RestController`, `@RequestMapping("/api/v1/storage")`, `@Tag(name = "storage")`
       - Constructor injection: `StorageService`

       **Endpoints:**

       a. **Upload file:**
          ```
          @PostMapping("/workspace/{workspaceId}/upload")
          @Operation(summary = "Upload a file to storage")
          @ApiResponses(
              ApiResponse(responseCode = "201", description = "File uploaded successfully"),
              ApiResponse(responseCode = "413", description = "File size exceeds limit"),
              ApiResponse(responseCode = "415", description = "Content type not allowed")
          )
          fun uploadFile(
              @PathVariable workspaceId: UUID,
              @RequestParam domain: StorageDomain,
              @RequestParam("file") file: MultipartFile
          ): ResponseEntity<UploadFileResponse>
          ```
          Returns `ResponseEntity.status(HttpStatus.CREATED).body(response)`

       b. **List files:**
          ```
          @GetMapping("/workspace/{workspaceId}/files")
          @Operation(summary = "List files in a workspace")
          fun listFiles(
              @PathVariable workspaceId: UUID,
              @RequestParam(required = false) domain: StorageDomain?
          ): ResponseEntity<FileListResponse>
          ```

       c. **Get file metadata:**
          ```
          @GetMapping("/workspace/{workspaceId}/files/{fileId}")
          @Operation(summary = "Get file metadata")
          fun getFile(
              @PathVariable workspaceId: UUID,
              @PathVariable fileId: UUID
          ): ResponseEntity<FileMetadata>
          ```

       d. **Generate signed URL:**
          ```
          @PostMapping("/workspace/{workspaceId}/files/{fileId}/signed-url")
          @Operation(summary = "Generate a signed download URL for a file")
          fun generateSignedUrl(
              @PathVariable workspaceId: UUID,
              @PathVariable fileId: UUID,
              @RequestBody(required = false) request: GenerateSignedUrlRequest?
          ): ResponseEntity<SignedUrlResponse>
          ```
          Pass `request?.expiresInSeconds` to service.

       e. **Delete file:**
          ```
          @DeleteMapping("/workspace/{workspaceId}/files/{fileId}")
          @Operation(summary = "Delete a file")
          @ApiResponse(responseCode = "204", description = "File deleted")
          fun deleteFile(
              @PathVariable workspaceId: UUID,
              @PathVariable fileId: UUID
          ): ResponseEntity<Void>
          ```
          Returns `ResponseEntity.noContent().build()`

       f. **Download via signed token (unauthenticated):**
          ```
          @GetMapping("/download/{token}")
          @Operation(summary = "Download a file using a signed URL token")
          @ApiResponses(
              ApiResponse(responseCode = "200", description = "File content streamed"),
              ApiResponse(responseCode = "403", description = "Token expired or invalid")
          )
          fun downloadFile(
              @PathVariable token: String,
              @RequestParam(required = false, defaultValue = "false") download: Boolean
          ): ResponseEntity<StreamingResponseBody>
          ```
          - Call `storageService.downloadFile(token)`
          - Set `Content-Type` header from result
          - Set `Content-Length` header from result
          - Set `Content-Disposition`: if `download=true` use `attachment; filename="originalFilename"`, else use `inline` (per research recommendation)
          - Return `StreamingResponseBody` that copies DownloadResult.content to output stream

       All endpoints except `downloadFile` follow the thin controller pattern -- delegate to service, return ResponseEntity. No business logic in the controller.

    2. Update `SecurityConfig.kt`:
       - Add `.requestMatchers("/api/v1/storage/download/**").permitAll()` BEFORE the `.anyRequest().authenticated()` line
       - This allows the signed URL download endpoint to work without JWT authentication (the HMAC token is the authorization)

    3. Add `@ApiResponses` and `@Operation` annotations to all endpoints per coding standards.
