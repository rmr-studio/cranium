Public method `introspect(ctx: PostgresCallContext, credentials: CredentialPayload): IntrospectionResult` (data class wrapping `SchemaIntrospectionResult` + `List<ForeignKeyMetadata>`). Internally:
      - Get pool, open connection.
      - Query 1 — INFORMATION_SCHEMA.tables WHERE table_schema = ctx.schema AND table_type = 'BASE TABLE'.

    `PostgresFetcher` — `@Component`. Constructor injects `WorkspaceConnectionPoolManager`, `ConnectorPoolProperties`, `PgTypeMapper`. Public method `fetch(ctx: PostgresCallContext, credentials: CredentialPayload, cursor: String?, limit: Int): RecordBatch`:
      - Resolve effective limit: `minOf(limit, props.defaultBatchSize)` so caller cannot over-request.
      - If `ctx.cursorColumn != null && ctx.cursorColumnIsTimestamp`: SQL = `SELECT * FROM "${ctx.schema}"."${ctx.tableName}" WHERE "${ctx.cursorColumn}" > ?::timestamptz ORDER BY "${ctx.cursorColumn}" ASC LIMIT ?`. Bind cursor param (parsed from String to Instant via `Instant.parse(cursor)`; null cursor = `1970-01-01T00:00:00Z`).
      - Else if `ctx.primaryKeyColumn != null`: SQL = `SELECT * FROM ... WHERE "${pk}" > ? ORDER BY "${pk}" ASC LIMIT ?`. Bind cursor as the pk's natural type (UUID / bigint / text) — pick from column metadata cached on the call (caller passes the column type via PostgresCallContext if needed; otherwise default to text comparison and let Postgres cast).
      - Else: throw AdapterCapabilityNotSupportedException("Table ${tableName} has neither a sync cursor nor a primary key — cannot fetch records").
      - Use `PreparedStatement` with `stmt.fetchSize = props.defaultBatchSize` and `connection.autoCommit = false` (server-side cursor — required for fetchSize to take effect on Postgres).
      - Iterate ResultSet, building each row's `Map<columnName, EntityAttributePrimitivePayload>`. For each column: fetch the typed JDBC value, run through `PgTypeMapper.toSchemaType(pgType, isPrimaryKey)` to get SchemaType; wrap value in `JsonValue.from(...)` (use `JsonValue.from` if it exists; otherwise build from primitive types directly per the JsonValue sealed hierarchy). For jsonb values, parse the PGobject's value string into a `JsonValue.Object` / `JsonValue.Array` via Jackson `objectMapper.readTree` + a JsonNode→JsonValue converter. For bytea, base64-encode into a `JsonValue.Object` `{"_bytea": "<base64>"}` shape.
      - Build SourceRecord: `externalId` = the value of the PK column as String (or the cursor column if no PK — corner case); `payload` = the column→payload map cast to `Map<String, Any?>` (the orchestrator treats payload as opaque Map per Phase 1 contract); `sourceMetadata` = mapOf("table" to tableName, "schema" to schema).
      - `nextCursor` = the cursor-column-or-PK value of the LAST row in the batch as String; `null` if records empty.
      - `hasMore` = `records.size == effectiveLimit` (heuristic — caller can issue another fetch).
